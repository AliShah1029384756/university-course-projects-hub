# Contributing

This repository is a public navigation hub for university project portfolio content.

## Contribution Scope

- Improve hub clarity and navigation
- Correct factual inconsistencies
- Improve project index quality and structure
- Fix links, metadata, and presentation issues

## How to Contribute

1. Open an issue for major structural changes.
2. Use focused pull requests with clear intent.
3. Keep factual statements aligned with source repositories.
4. Preserve the 16-project consistency rule for active content.

## Quality Rules

- Use precise, verifiable wording.
- Avoid inflated claims.
- Keep sections easy to scan for reviewers.
- Ensure all external links are relevant and active.
